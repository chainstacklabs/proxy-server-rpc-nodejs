Check Your Balance

This webpage uses a Chainstack Ethereum RPC endpoint to retrieve the balance of an address.

The idea of this app is to teach you how to properly protect your API keys!

Inspecting the source code, you will notice that you won't find any API key or endpoint URL in the front end; you will find instead a connection to a back-end proxy server where the endpoint URL is safely stored. This is a simple way to protect your endpoint URL.
